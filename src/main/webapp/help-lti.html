Help for Canvas Course Manager

Note: This documentation only applies to course instructors and Canvas account and sub-account admins; the ITS Service Center should consult separate documentation in their Knowledgebase.

Introduction

The Canvas Course Management tool allows Canvans sub-account admins, UM instructors and users in specific roles to perform certain functions.

CCM is not visible to students (it does not appear in the left hand navigation for them). You can verify this if you like by checking Student View for your course.

Only UM uniqname holders can use CCM.

  • Instructors, account and sub-account admins, and other users with certain course roles:
    • Add non-UM external users to a Canvas course
    • Add sections to an existing Canvas course
    • Gradebook CSV Formatter will convert a downloaded Canvas gradebook CSV to an MPathways formatted version and download it to your computer.
  • Bulk user, section and group tools for account/sub-account admins:
    • Add multiple sections to a course via CSV upload
    • Add multiple users to mulitple sections via CSV upload or web form (data grid)
    • Add groups and users via CSV upload or web form (data grid)

Adding non-UM external users to your Canvas course

To add external users, you must have an appropriate course role in Canvas (Teacher, TA, Assistant or Designer).

  1. Click Add non-UM users.
  2. In the box that opens, type the full email address of the external user, e.g., exampleuser@gmail.com
  3. Check the box for the section you would like to add the external user to.
  4. On the right, choose a role for the user. Then click Next.
  5. The first time an external user is added to Canvas, you'll be asked for his or her first and last name. If the user has not previously used Canvas at UM, he or she will be sent an email with instructions on how to create a password and with a link to Canvas.

    No other emails are sent automatically; some instructors choose to send an individual email to the users letting them know they've been added to the course.

  6. You'll see a confirmation that the user was added. You can then choose to add more users or click Done.

To delete a user who was added with the Course Management tool, use the Canvas People tool.

Adding sections to your Canvas course

To add a section to a Canvas course, you must be listed as the official instructor (primary, secondary, grad student instructor, or faculty grader) in MPathways.

  1. Click Add Sections.
  2. In "Your Other Courses," click Choose Sections to Add to the right of a course.
  3. Click Add to Course to add a section.
  4. Before saving an added section, you have the option to return it to the original course. Once saved, it's still possible to move it back to the original course (or to another course) by using the Course Management tool while you are in the other course.

    Note that you cannot use the Course Management tool to completely remove a section from all of your courses. To do that, you will need to contact 4HELP@umich.edu

  5. When finished, click Save Sections and then Done.

Gradebook CSV Formatter

  1. Download your course gradebook in Canvas
  2. Verify the data
  3. In UM-Course Manager, click the CSV Gradebook Formatter link
  4. Drag your exported CSV into the tool or click the upload button
  5. The tool will convert your file to an MPathways compatible format and download it back to your computer.

Bulk Course Tools for Account Admins

Click the Account Admin Tools link at the top of the UM Course Manager Tool.

Adding multiple sections to a course:

  1. Create a CSV file with two columns: id_prefix, name

    • id_prefix: a prefix for each section SIS ID; must be at least three characters, alpha-numeric, no spaces
    • name: section name; must be at least five characters, alpha-numeric, spaces permitted

      TIP: Make the id_prefix value meaningful to you. It will be helpful when adding users to the sections. For example, if you are creating sections called Section A, Section B, etc. a good prefix might be sectA, sectB, etc.

      Example CSV data as viewed in Excel:
      id_prefix
      		 name
      sectA
      		 Section A
      sectB
      		 Section B
      sectC
      		 Section C


  2. Select Add Multiple Sections to course via CSV upload
  3. Select your file and wait while the tool validates the data. Address any errors by correcting your CSV and re-uploading.
  4. You will receive an email when the process is complete
  5. Visit the course settings > sections tab to verify section creation.

Adding multiple users to sections via CSV

This tool can be used with any sections with SIS IDs only.
  1. Create a CSV file with these three fields: user_id, role, section_id
    • user_id: user's SIS ID or non-UM email address (non-UM users must already be added to Canvas).
    • role: typically student, but the other Canvas roles are also supported
    • section_id: The section SIS ID. Click the link in the tool to see valid section IDs.

      Example CSV data as viewed in Excel:
      user_id
      		 role
      		 section_id
      99999991
      		 student
      		 sectA-ccmS143225
      99999992
      		 student
      		 sectA-ccmS143225
      99999993
      		 student
      		 sectB-ccmS143225
      some-non-um-user@mailinator.com
      		 student
      		 sectB-ccmS143225
      99999995
      		 student
      		 sectC-ccmS143225


  2. Select your file and wait while the tool validates the data. Address any errors by correcting your CSV and re-uploading.

    TIP: This tool only validates that the data is properly formatted for the sections in the course. If you attempt to add a users that is not in Canvas, you will recieve an error message in the email confirmation.

  3. You will receive an email confirmation when the process is complete.
  4. Visit the course settings > sections tab to verify users have been added

Add new groups and course members to course via CSV

Only one groupset per CSV upload.
  1. Create a CSV with these fields: groupset, name, user_id
    • groupset: A name for the Canvas Groupset. This value must be the same in each row of your CSV
    • name: The name of each group
    • user_id: the course member's SIS ID or non-UM email address

      Example CSV data as viewed in Excel:
      groupset
      		 name
      		 user_id
      Course Teams
      		 Team 1
      		 99999991
      Course Teams
      		 Team 1
      		 99999992
      Course Teams
      		 Team 2
      		 99999993
      Course Teams
      		 Team 2
      		 some-non-um-user@mailinator.com
      Course Teams
      		 Team 3
      		 99999995

  2. Select your file and wait while the tool validates the data. Address any errors by correcting your CSV and re-uploading

    TIP: The tool doesn't verify that the users are in your course. If you add someone who isn't in the course, you will see an error message in the email confirmation

  3. You will receive an email confirmation when the process is complete
  4. Visit your course People tool to view the groups

Add multiple users to course sections through data grid

This is a web-form tool.

For each user:
  1. Enter or copy/paste the user's SIS ID or non-UM email address in the user_id field
  2. Select a section and role for each user
  3. Click button to add more rows if needed
  4. Click Submit when all data is entered.
  5. You will receive an email confirmation when the process is complete
  6. Visit the course settings > sections tab to verify users have been added

Add new groups and course members through data grid.
Same rules apply as for the CSV upload version of this tool.
